# 🧠 Smart Notes AI

A full-stack MERN application that allows users to create, manage, and enhance notes using AI.

Smart Notes AI is a CRUD-based Notes Management System with authentication and AI-powered content enhancement. It helps users improve their writing with a single click.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- User Login
- JWT-based Authentication
- Protected Routes

### 📝 Notes Management (CRUD)
- Create Note
- Read All Notes
- Update Note
- Delete Note
- Search Notes
- Mark as Important ⭐

### 🤖 AI Integration
- Enhance note content using AI
- Summarize note
- Improve writing clarity
- Automatic note update after AI response

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt.js
- AI API Integration
